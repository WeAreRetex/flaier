# Installation

## Prerequisites

- Vue 3.5+
- Vite 8+ (or any bundler that handles `.vue` SFCs)

## Install the package

:::code-group

```bash [pnpm]
pnpm add flow-narrator
```

```bash [npm]
npm install flow-narrator
```

:::

## Import the stylesheet

```ts [main.ts]
import "flow-narrator/style.css";
```

In a Nuxt app, add it via `nuxt.config.ts`:

```ts [nuxt.config.ts]
export default defineNuxtConfig({
  css: ["flow-narrator/style.css"],
});
```

## Register the component

### Vue (global registration)

```ts [main.ts]
import { createApp } from "vue";
import { FlowNarrator } from "flow-narrator";
import "flow-narrator/style.css";
import App from "./App.vue";

const app = createApp(App);
app.component("FlowNarrator", FlowNarrator);
app.mount("#app");
```

### Vue (local registration)

```vue [App.vue]
<script setup lang="ts">
import { FlowNarrator } from "flow-narrator";
</script>

<template>
  <FlowNarrator src="./flow-specs/manifest.json" />
</template>
```

## Render your first flow

Point `src` at a local spec file or manifest. For development, drop a spec in `public/flow-specs/` so Vite serves it statically.

```vue
<FlowNarrator src="/flow-specs/manifest.json" :interval="3000" />
```
