# Installation

## Prerequisites

- Vue 3.5+
- Vite 6+ (or any bundler that handles `.vue` SFCs)

## Install the package

:::code-group

```bash [pnpm]
pnpm add flow-narrator
```

```bash [npm]
npm install flow-narrator
```

:::

## Import the stylesheet

```ts [main.ts]
import "flow-narrator/style.css";
```

In a Nuxt app, add it via `nuxt.config.ts`:

```ts [nuxt.config.ts]
export default defineNuxtConfig({
  css: ["flow-narrator/style.css"],
});
```

## Register the component

### Vue (global registration)

```ts [main.ts]
import { createApp } from "vue";
import { FlowNarrator } from "flow-narrator";
import "flow-narrator/style.css";
import App from "./App.vue";

const app = createApp(App);
app.component("FlowNarrator", FlowNarrator);
app.mount("#app");
```

### Vue (local registration)

```vue [App.vue]
<script setup lang="ts">
import { FlowNarrator } from "flow-narrator";
</script>

<template>
  <FlowNarrator src="./flow-specs/manifest.json" />
</template>
```

## Nuxt / Docus integration

If you are using Nuxt, Docus, or Nuxt Content, install the companion module:

:::code-group

```bash [pnpm]
pnpm add flow-narrator @flow-narrator/nuxt
```

```bash [npm]
npm install flow-narrator @flow-narrator/nuxt
```

:::

```ts [nuxt.config.ts]
export default defineNuxtConfig({
  modules: ["@flow-narrator/nuxt"],
});
```

The module auto-registers three global wrappers:

- `FlowNarratorClient` for app pages and regular Nuxt components.
- `FlowNarratorDemo` for fixed-height demo embeds.
- `FlowNarratorMdc` for `.md` and `.mdc` docs content with fullscreen support.

```vue
<template>
  <FlowNarratorClient src="/flow-specs/manifest.json" :auto-play="true" :interval="2800" />
</template>
```

## Fixed-height and fullscreen embeds

Use `FlowNarratorPanel` when you want a constrained canvas with a built-in fullscreen toggle. The docs site and Slidev example in this repo both use this wrapper.

```vue
<script setup lang="ts">
import { FlowNarratorPanel } from "flow-narrator";
import spec from "./story-architecture.flow.json";
</script>

<template>
  <FlowNarratorPanel :src="spec" height="min(72vh, 760px)" :fullscreen-enabled="true" />
</template>
```

## Render your first flow

Point `src` at a local spec file or manifest. For development, drop a spec in `public/flow-specs/` so Vite serves it statically.

```vue
<FlowNarrator src="/flow-specs/manifest.json" :interval="3000" />
```
