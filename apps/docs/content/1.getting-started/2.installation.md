# Installation

## Prerequisites

- Vue 3.5+
- Vite 6+ (or any bundler that handles `.vue` SFCs)

## Install the package

:::code-group

```bash [pnpm]
pnpm add @flaier/core
```

```bash [npm]
npm install @flaier/core
```

:::

## Import the stylesheet

```ts [main.ts]
import "@flaier/core/style.css";
```

In a Nuxt app, add it via `nuxt.config.ts`:

```ts [nuxt.config.ts]
export default defineNuxtConfig({
  css: ["@flaier/core/style.css"],
});
```

## Register the component

### Vue (global registration)

```ts [main.ts]
import { createApp } from "vue";
import { Flaier } from "@flaier/core";
import "@flaier/core/style.css";
import App from "./App.vue";

const app = createApp(App);
app.component("Flaier", Flaier);
app.mount("#app");
```

### Vue (local registration)

```vue [App.vue]
<script setup lang="ts">
import { Flaier } from "@flaier/core";
</script>

<template>
  <Flaier src="./flow-specs/manifest.json" />
</template>
```

## Nuxt / Docus integration

If you are using Nuxt, Docus, or Nuxt Content, install the companion module:

:::code-group

```bash [pnpm]
pnpm add @flaier/core @flaier/nuxt
```

```bash [npm]
npm install @flaier/core @flaier/nuxt
```

:::

```ts [nuxt.config.ts]
export default defineNuxtConfig({
  modules: ["@flaier/nuxt"],
});
```

The module auto-registers three global wrappers:

- `FlaierClient` for app pages and regular Nuxt components.
- `FlaierDemo` for fixed-height demo embeds.
- `FlaierMdc` for `.md` and `.mdc` docs content with fullscreen support.

```vue
<template>
  <FlaierClient src="/flow-specs/manifest.json" :auto-play="true" :interval="2800" />
</template>
```

## Register custom nodes

Use the same node definition object in two places:

- `createFlaierCatalog({ nodes })` so AI prompts/spec generation know about the new node type.
- `<Flaier :nodes="nodes" />` or `registerFlaierNodes(nodes)` so the renderer can display it.

```ts [custom-nodes.ts]
import { z } from "zod";
import { createFlaierCatalog, sourceAnchorSchema, edgeTransitionSchema } from "@flaier/core";
import CalloutNode from "./CalloutNode.vue";

export const nodes = {
  CalloutNode: {
    props: z.object({
      label: z.string(),
      body: z.string(),
      sourceAnchor: sourceAnchorSchema.optional(),
      transitions: z.array(edgeTransitionSchema).optional(),
    }),
    description: "Narrative callout block for custom flow annotations",
    component: CalloutNode,
  },
};

export const catalog = createFlaierCatalog({ nodes });
```

```vue [App.vue]
<script setup lang="ts">
import { Flaier } from "@flaier/core";
import { nodes } from "./custom-nodes";
</script>

<template>
  <Flaier src="/flow-specs/manifest.json" :nodes="nodes" />
</template>
```

In Nuxt, register them once in a plugin so `FlaierClient`, `FlaierDemo`, and `FlaierMdc` all see the same nodes:

```ts [plugins/flaier-nodes.ts]
import { nodes } from "~/custom-nodes";

export default defineNuxtPlugin(() => {
  registerFlaierNodes(nodes);
});
```

## Customize styles

The shipped CSS exposes theme tokens on `.flaier`. Override them in your app stylesheet to change the renderer without forking the package:

```css [app.css]
.flaier {
  --color-primary: oklch(0.72 0.16 20);
  --fn-edge-active: oklch(0.72 0.16 20);
}

.fn-panel {
  --fn-panel-toggle-bg: rgba(17, 24, 39, 0.92);
  --fn-panel-toggle-hover-border: rgba(244, 114, 182, 0.7);
}

.fn-node[data-node-type="architecture"][data-node-kind="database"] {
  border-style: solid;
}
```

## Fixed-height and fullscreen embeds

Use `FlaierPanel` when you want a constrained canvas with a built-in fullscreen toggle. The docs site and Slidev example in this repo both use this wrapper.

```vue
<script setup lang="ts">
import { FlaierPanel } from "@flaier/core";
import spec from "./story-architecture.flow.json";
</script>

<template>
  <FlaierPanel :src="spec" height="min(72vh, 760px)" :fullscreen-enabled="true" />
</template>
```

## Render your first flow

Point `src` at a local spec file or manifest. For development, drop a spec in `public/flow-specs/` so Vite serves it statically.

```vue
<Flaier src="/flow-specs/manifest.json" :interval="3000" />
```
