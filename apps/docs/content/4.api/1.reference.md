# API Reference

## Components

### `<FlowNarrator>`

The top-level component. Accepts a spec, manifest, or URL and renders the appropriate viewer.

```vue
<FlowNarrator src="./flow-specs/manifest.json" :auto-play="true" :interval="3000" />
```

| Prop       | Type                 | Default | Description                                                      |
| ---------- | -------------------- | ------- | ---------------------------------------------------------------- |
| `src`      | `FlowNarratorSource` | —       | Required. Spec object, manifest object, or file path/URL string. |
| `autoPlay` | `boolean`            | `false` | Start playing automatically on load.                             |
| `interval` | `number`             | `3000`  | Milliseconds between auto-advance steps (narrative mode).        |

`FlowNarratorSource` is `FlowNarratorSpec | FlowNarratorManifest | string`.

## Composables

### `useShiki()`

Access the shared Shiki highlighter instance. Useful if you need to highlight code outside of a `CodeNode`.

```ts
import { useShiki } from "flow-narrator";

const { highlight } = useShiki();
const html = await highlight("const x = 1", "typescript");
```

### `useTimeline()`

Read and control the timeline state from a parent component.

```ts
import { useTimeline } from "flow-narrator";

const { currentStep, totalSteps, next, prev, play, pause, isPlaying } = useTimeline();
```

| Return        | Type                  | Description                        |
| ------------- | --------------------- | ---------------------------------- |
| `currentStep` | `Ref<number>`         | Index of the active node.          |
| `totalSteps`  | `ComputedRef<number>` | Total number of nodes in the flow. |
| `next()`      | `() => void`          | Advance to the next node.          |
| `prev()`      | `() => void`          | Go back to the previous node.      |
| `play()`      | `() => void`          | Start autoplay.                    |
| `pause()`     | `() => void`          | Stop autoplay.                     |
| `isPlaying`   | `Ref<boolean>`        | Whether autoplay is active.        |

### `useFlowNarratorRuntime()`

Access export and runtime utilities.

```ts
import { useFlowNarratorRuntime } from "flow-narrator";

const { exportPng, exportPdf } = useFlowNarratorRuntime();
```

| Return        | Type                  | Description                 |
| ------------- | --------------------- | --------------------------- |
| `exportPng()` | `() => Promise<void>` | Export full diagram as PNG. |
| `exportPdf()` | `() => Promise<void>` | Export full diagram as PDF. |

## Catalog & Registry

flow-narrator exposes the json-render catalog and registry for advanced use cases where you need to compose specs programmatically.

```ts
import { catalog, registry } from "flow-narrator";
```

- `catalog` — The json-render catalog describing all flow-narrator node schemas.
- `registry` — The json-render component registry mapping type names to Vue components.
