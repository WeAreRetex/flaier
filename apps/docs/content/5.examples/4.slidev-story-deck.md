# Slidev Story Deck

This repo includes a Slidev example app in `apps/slides` that embeds `FlowNarratorPanel` directly inside deck slides.

**Entry files:** `apps/slides/slides.md`, `apps/slides/components/ArchitectureFlow.vue`, `apps/slides/flow-specs/story-architecture.flow.json`

## Run the example

```bash
pnpm slides:dev
```

To build the deck:

```bash
pnpm slides:build
```

## Presentation-friendly spec settings

The Slidev architecture spec trims standalone viewer chrome so the diagram fits a deck canvas cleanly:

```json
{
  "type": "FlowTimeline",
  "props": {
    "mode": "architecture",
    "themeMode": "document",
    "showHeaderOverlay": false,
    "showExportControls": false,
    "showThemeToggle": true,
    "showArchitectureInspector": true,
    "defaultArchitectureInspectorOpen": false,
    "showArchitectureInspectorToggleText": false
  }
}
```

## Why `FlowNarratorPanel`

`FlowNarratorPanel` gives the deck a fixed-height canvas and a fullscreen toggle without extra Slidev-specific plumbing. That makes it a good default when the renderer lives inside docs pages, demos, or slides instead of occupying the full browser viewport.
