# Branching Flow

A narrative flow that demonstrates decision nodes and diverging paths. The webhook payload is validated, then routed to either a delivery path or a quarantine path based on the result.

**Spec:** `branching.flow.json` · **Mode:** narrative · **Nodes:** 7

## When to use

Branching flows work well for:

- Error handling and retry logic
- Feature flag routing
- Approval workflows
- Conditional processing pipelines
