# Architecture Map

An architecture-mode flow that renders all nodes simultaneously as a topology map. Click any node to inspect its metadata. This mode is designed for system design documents and architecture reviews.

## When to use

Architecture maps work well for:

- System design documents
- Service dependency visualization
- Infrastructure topology overviews
- Architecture decision records (ADRs)
